# Media Files

Tambahkan file berikut ke folder `public/`:

## Videos
- `/videos/food-background.mp4` - Video makanan untuk background hero (rekomendasikan durasi 30 detik, loop-ready)
- `/videos/reel-1.mp4` - Video reel 1
- `/videos/reel-2.mp4` - Video reel 2
- `/videos/reel-3.mp4` - Video reel 3

## Audio
- `/audio/background.mp3` - Audio background dummy (gunakan musik latar ambient, durasi fleksibel)

Anda dapat menggunakan platform berikut untuk mendapatkan media:
- **Pexels Videos**: https://www.pexels.com/videos/
- **Pixabay**: https://pixabay.com/videos/
- **Free Music Archive**: https://freemusicarchive.org/

Atau buat video sendiri menggunakan:
- CapCut (mudah dan cepat)
- OBS Studio (gratis dan powerful)
